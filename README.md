# PDBDownloader
Downloads PDB files from msdl.microsoft.com

![image](https://github.com/Pinguin2001/PDBDownloader/assets/75797743/876e8892-1a70-40bf-9107-cbd0d0228c4f)

# Usage

```batch
PDBDownloader <PE File> <PDB output file>
```
